REST API for Developers

How to use REST API

To extract an individual user's IPPT data, replace {username} with the username of the user you want to access.

https://trainify-production.up.railway.app/api/user/{username}

To access sample user's IPPT data, click the link shown below.

https://trainify-production.up.railway.app/api/user/test

To access all user's IPPT data, click the link shown below.

https://trainify-production.up.railway.app/api/alluser
